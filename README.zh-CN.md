# Marketing Plan Bundle

市场营销计划管理模块，提供以下功能：

- 创建和管理营销计划
- 流程节点管理
- 状态追踪
- 条件判断
- 延时管理
- 用户进度管理

## 安装

```bash
composer require tourze/marketing-plan-bundle
```
